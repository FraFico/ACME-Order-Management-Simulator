# PDF di input

Metti qui i file PDF degli ordini/documenti da processare.

Esempio d'uso:

```bash
python src/main.py data/pdf_input/ordine_cliente_A.pdf
```

Questa cartella è vuota nel repository: i PDF reali degli ordini contengono
dati potenzialmente sensibili (nomi cliente, importi) e non vengono versionati.
Aggiungi qui i tuoi file di test in locale.
